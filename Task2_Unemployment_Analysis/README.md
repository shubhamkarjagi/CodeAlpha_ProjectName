# Task 2: Unemployment Analysis with Python

## Description
This task focuses on analyzing unemployment rate data using Python.
The objective is to understand unemployment trends, seasonal patterns,
and the impact of Covid-19 on unemployment rates.

## Dataset
The dataset contains unemployment-related information such as:
- Unemployment rate
- Date / Time period
- Region or Area (if applicable)

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Trend analysis
- Seasonal pattern identification
- Covid-19 impact analysis on unemployment rates

## Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Visualizations
- Line plots to show unemployment trends
- Comparative plots for pre and post Covid-19 analysis
- Seasonal trend visualizations

## Outcome
The analysis provides meaningful insights into unemployment patterns
and helps understand how economic events like Covid-19 affected employment.
These insights can support economic and social policy decisions.
